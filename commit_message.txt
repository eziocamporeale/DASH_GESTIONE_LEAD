🎉 Initial commit: DASH_GESTIONE_LEAD - Dashboard completa per gestione lead

## 📋 Descrizione del Progetto
Dashboard completa per la gestione dei lead aziendali con funzionalità avanzate di categorizzazione, assegnazione ruoli, tracking del processo di vendita e analisi delle performance.

## ✅ Funzionalità Implementate

### 🔐 Sistema Autenticazione
- Login/logout funzionante con gestione sessioni
- Sistema ruoli e permessi granulare
- Credenziali di default: admin/admin123

### 📊 Dashboard Principale
- Metriche KPI principali con grafici interattivi
- Grafici lead per stato e fonte
- Lista task recenti
- Design moderno con CSS personalizzato

### 🗄️ Database Completo
- 16 tabelle per gestione completa del sistema
- Schema ottimizzato con indici appropriati
- Dati di default per testing
- Backup automatico

### 👥 Modulo Gestione Lead
- Form inserimento nuovo lead con validazione
- Tabella lead con filtri avanzati
- Categorizzazione automatica
- Assegnazione utenti
- Analytics lead dettagliate

### ✅ Modulo Task Management
- Form creazione task con priorità
- Board Kanban interattiva
- Tracking progresso task
- Notifiche scadenze
- Filtri e ricerca avanzata

### 👤 Modulo Gestione Utenti
- Form creazione e modifica utenti
- Gestione ruoli (Admin, Manager, Setter, Closer, Viewer)
- Gestione dipartimenti
- Sistema permessi granulare

### 📞 Modulo Contatti
- Template email/SMS personalizzabili
- Sequenze di contatto automatiche
- Storico contatti dettagliato
- Metriche efficacia sequenze

### ⚙️ Modulo Impostazioni
- Configurazione sistema
- Gestione backup
- Log applicazione
- Personalizzazione dashboard

## 🏗️ Architettura Tecnica

### Frontend
- Streamlit per interfaccia web
- Plotly per grafici interattivi
- CSS personalizzato per design moderno
- Layout responsive per mobile/desktop

### Backend
- Python 3.8+ come linguaggio principale
- SQLite per database locale
- Sistema modulare e scalabile
- Gestione errori robusta

### Database
- Schema normalizzato con 16 tabelle
- Indici ottimizzati per performance
- Foreign keys per integrità dati
- Backup automatico
